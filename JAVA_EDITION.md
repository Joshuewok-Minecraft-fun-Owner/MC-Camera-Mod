# Java Edition vs Bedrock Edition

## Comparison

| Feature | Java | Bedrock |
|---------|------|---------|
| **Platform** | PC (Linux/Mac/Windows) | Console, Mobile, PC |
| **Mod Format** | JAR (Fabric/Forge) | .mcaddon |
| **Language** | Java | JavaScript |
| **Camera System** | ✅ Full | ✅ Full |
| **Privacy Tools** | ✅ Full | ✅ Full |
| **Optimization** | ✅ Full | ✅ Full |
| **Auto-Update** | ✅ Yes | ⏳ Manual |
| **UI Method** | Screens/Keybinds | Custom Menus |

## Installation Comparison

### Java Edition (Fabric)
```
1. Download fabric loader
2. Install fabric-api mod
3. Place CreatorTools.jar in mods folder
4. Launch Minecraft
5. Press P to open menu
```

### Bedrock Edition
```
1. Download CreatorTools.mcaddon
2. Double-click to install
3. Enable in world settings
4. Type !tools in chat
5. Navigate menu
```

## Key Differences

### Java Edition Features
- ✅ Automatic update checker
- ✅ Keybind customization
- ✅ GUI-based screens
- ✅ Faster mod loader updates
- ✅ Compatible with OptiFine
- ✅ Console-based debugging

### Bedrock Edition Features

- ✅ Easier installation (.mcaddon)
- ✅ Built-in UI system

## Which Should You Use?

### Use Java Edition If:
- ✅ You play on PC (Windows/Mac/Linux)
- ✅ You want automatic updates
- ✅ You use other Fabric mods
- ✅ You want maximum customization

### Use Bedrock Edition If:
- ✅ You play on console or mobile
- ✅ You prefer simpler installation
- ✅ You use Minecraft Realms
- ✅ You want built-in cross-platform features
