## 🚀 Getting Started

### Step 1: Installation (2 minutes)
1. Copy `CreatorToolsBP` to your Minecraft `behavior_packs` folder
2. Copy `CreatorToolsRP` to your Minecraft `resource_packs` folder
3. Enable both packs in world settings (Behavior Pack first)

### Step 2: Using the Pack
1. Open chat and type: `!tools`
2. Select the tool you want (Camera, Privacy, or Optimization)
3. Adjust settings or enable features

### Step 3: Explore Commands
```
!tools              # Open main menu
!camera             # Camera tools only
!privacy            # Privacy settings only
!optimize           # Optimization settings only
!preset [name]      # Apply preset directly
```
